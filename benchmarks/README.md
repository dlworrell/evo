# EVO Benchmarks

The benchmark suite will measure:

- convergence quality,
- convergence speed,
- population diversity,
- sensitivity to random seed,
- runtime,
- memory use,
- scalability of parallel evaluation,
- and robustness on deceptive or multimodal fitness landscapes.

Initial benchmarks should include simple analytic functions, constrained optimization, permutation problems, and the repository-readiness scoring pilot.

Results must follow the evidence requirements in `docs/benchmarks.md`.
